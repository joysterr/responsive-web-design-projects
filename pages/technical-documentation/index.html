introduction

Swift is a powerful and intuitive programming language developed by Apple for building iOS, macOS, watchOS, and tvOS applications. Introduced in 2014, Swift is designed to be fast, safe, and expressive, providing developers with a modern and efficient language for creating a wide range of software.


Key Features
  • Safety: Swift eliminates common programming errors by enforcing type safety and providing optionals to handle the absence of a value.

  • Performance: With a focus on optimization, Swift delivers high performance for both development and runtime, making it suitable for resource-intensive applications.

  • Expressiveness: Swift syntax is concise and expressive, allowing developers to write clean and maintainable code.

  • Interoperability: Swift seamlessly integrates with Objective-C, enabling developers to use existing codebases alongside new Swift projects.

  • Concurrency Support: Swift introduces modern concurrency features, such as async/await, to simplify and enhance the handling of asynchronous operations, improving the responsiveness of applications.
variables
Declaring Variables

In Swift, variables are mutable, meaning their values can be changed after they are initially set. They are declared using the var keyword.

var myVariable: Int = 42
Constants

Constants are declared using the let keyword and are immutable, meaning their values cannot be changed once assigned.

let pi: Double = 3.14159
Type Inference

Swift uses type inference to automatically deduce the type of a variable or constant, reducing the need for explicit type annotations.

let message = "Hello, Swift!"
functions
Function Declaration

Functions in Swift are defined using the func keyword. Parameters and return types can be explicitly specified.

func greet(name: String) -> String {
  return "Hello, \(name)!"
}
External and Internal Parameter Names

Swift allows the use of external parameter names to improve the readability of function calls while maintaining concise internal parameter names.

func calculateSum(of numbers: [Int]) -> Int {
  return numbers.reduce(0, +)
}
control flow
Conditional Statements

Swift supports traditional if, else if, and else statements for implementing conditional logic.

let temperature = 25
if temperature > 30 {
  print("It's hot outside!") } else if temperature >= 20 {
  print("The weather is pleasant.")
} else {
  print("It's a bit chilly.")
}
Loops

Swift provides for-in loops for iterating over collections and ranges, as well as while and repeat-while loops.

for i in 1...5 {
  print("Count: \(i)")
}

var countdown = 3
while countdown > 0 {
  print("T-minus \(countdown)")
  countdown -= 1
}
optionals
Optionals

Optionals in Swift represent the absence of a value. They are declared using the Type? syntax.

var optionalNumber: Int? = nil
Error Handling

Swift uses the throws keyword to indicate that a function can throw an error. Errors are caught using the do, try, catch pattern.

enum FileError: Error {
  case fileNotFound
  case permissionDenied
}

func readFile() throws {
  // ...
}

do {
  try readFile()
} catch FileError.fileNotFound {
  print("File not found.")
} catch FileError.permissionDenied {
  print("Permission denied.")
} catch {
  print("An unexpected error occurred.")
}
